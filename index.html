SQL Package Summary

This page summarizes common Oracle SQL built-in packages.

  • DBMS_SQL: Executes dynamic SQL from PL/SQL.
  • UTL_FILE: Read/write text files from the server.
  • DBMS_OUTPUT: Shows output from PL/SQL blocks (like print).
  • UTL_SMTP: Send emails using SMTP.
  • DBMS_CRYPTO: Hashing & encryption functions.
